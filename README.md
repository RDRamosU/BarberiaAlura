# Baron Barbershop — Landing Page

Landing page para una barbería moderna construida con HTML y CSS puro. Desarrollada para practicar diseño web, tipografía, layout con CSS Grid y Flexbox, y principios de diseño minimalista.

---

## ¿Qué es?

Baron Barbershop es una página web estática que simula el sitio oficial de una barbería clásica. El diseño sigue una paleta blanco y negro con tipografía serif elegante, combinando técnicas modernas de layout con una estética atemporal.

## Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura semántica de la página |
| CSS3 | Diseño, tipografía y layout |
| CSS Grid | Layout de servicios, galería y contacto |
| Flexbox | Nav, footer y elementos internos |
| Google Fonts | Playfair Display y DM Sans |

---

## Conceptos practicados

- **Tipografía** — combinación de fuente serif (Playfair Display) y sans-serif (DM Sans)
- **CSS Grid** — layout de servicios en 3 columnas, galería con `grid-row: span 2`
- **Flexbox** — navegación, footer y grids secundarios
- **Variables CSS** — sistema de colores y fuentes centralizado con `:root`
- **Diseño responsivo** — media queries para móvil y tablet
- **Navbar fija** — con `position: fixed` y `backdrop-filter: blur`
- **Paleta monocromática** — diseño en blanco y negro clásico

---

## Secciones

- **Hero** — nombre de la barbería con tipografía grande y botón de llamado a acción
- **Servicios** — 6 servicios en grid de 3 columnas con precios
- **Sobre nosotros** — historia y estadísticas de la barbería
- **Galería** — grid de placeholders para fotos del trabajo
- **Contacto** — dirección, teléfono y horarios
- **Footer** — logo y créditos

## Instalación

No requiere instalación ni dependencias. Clona el repositorio y abre el archivo directamente en tu navegador:

```bash

cd BarberiaAlura
```

Abre `index.html` en el navegador (favorito).
